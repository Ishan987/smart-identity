**Smart Identity Pro**
Smart Identity Pro is a Python-based desktop application designed to manage and verify identity documents efficiently. 
The application allows users to upload, scan, extract, and manage identity information from documents such as ID cards, passports, and other official records.

**Technologies Used**
1.Python
2.PyQt5 / Tkinter – GUI Development
3.OpenCV (cv2) – Image processing
4.PyMuPDF (fitz) – PDF processing
